# 第 1 週今日 Node.js 練習題：建立你的第一個模組系統

## 題目說明

在後端開發中，程式碼的組織與模組化是非常重要的基礎。Node.js 採用 CommonJS 模組系統，讓你能將不同功能拆分成獨立檔案，透過 `require` 載入、`module.exports` 匯出。本題將帶你從零開始建立一個簡單的「個人資料計算工具」，練習 Node.js 的模組拆分概念，並使用 npm 初始化專案。這是往後建立 Express API 的重要根基——每個 route handler、middleware、資料庫邏輯都會用到模組化設計。

## 技術要求

- 概念：Node.js 執行環境、`require`、`module.exports`、`npm init`
- 需要的套件：無（僅使用 Node.js 內建功能）

## 具體任務

1. 在專案資料夾內執行 `npm init -y`，建立 `package.json`
2. 建立 `mathUtils.js`，匯出以下三個函式：
   - `add(a, b)` — 回傳兩數相加結果
   - `subtract(a, b)` — 回傳兩數相減結果
   - `average(numbers)` — 接受一個數字陣列，回傳平均值
3. 建立 `profileCalculator.js`，匯出一個函式 `generateReport(name, scores)`：
   - 使用 `require` 載入 `mathUtils.js`
   - `scores` 是一個數字陣列（代表考試分數）
   - 回傳一個物件，包含 `name`、`highest`、`lowest`、`average` 四個欄位
4. 建立 `index.js` 作為程式進入點：
   - 載入 `profileCalculator.js`
   - 呼叫 `generateReport`，傳入你自己的名字和一組分數
   - 用 `console.log` 將結果輸出到終端機

## 範例

```js
// mathUtils.js
module.exports = {
  add: (a, b) => a + b,
  subtract: (a, b) => a - b,
  average: (numbers) => numbers.reduce((sum, n) => sum + n, 0) / numbers.length,
};

// profileCalculator.js
const { add, subtract, average } = require('./mathUtils');

function generateReport(name, scores) {
  return {
    name,
    highest: Math.max(...scores),
    lowest: Math.min(...scores),
    average: average(scores),
  };
}

module.exports = { generateReport };

// index.js
const { generateReport } = require('./profileCalculator');

const result = generateReport('小明', [85, 92, 78, 90, 88]);
console.log(result);

// 預期輸出：
// { name: '小明', highest: 92, lowest: 78, average: 86.6 }
```

## 提示

- `module.exports` 可以匯出物件（包含多個函式），也可以直接匯出單一函式
- `require('./mathUtils')` 中的 `./` 代表「同一目錄」，不可省略
- 執行方式：在終端機輸入 `node index.js`
- `Math.max(...scores)` 使用展開運算子將陣列展開為個別參數

## 學習重點

模組化設計是後端 API 的骨架。當你日後使用 Express 建立 API 時，每個路由（如 `/users`、`/products`）會被拆分成獨立的 router 檔案，資料庫查詢邏輯會放在獨立的 service 或 model 層，全部透過 `require` 串聯起來。現在練習的 `module.exports` / `require` 模式，就是那個架構的最小單元。
