# 第 1 週今日 Node.js 練習題：模組系統與 npm 初始化

## 題目說明

學習 Node.js 最核心的模組系統，包含如何用 `require` 引入模組、用 `module.exports` 匯出功能，以及用 `npm init` 建立專案。這是開發任何 Node.js 應用程式的基礎。

## 技術要求

- **概念**：CommonJS 模組系統（`require` / `module.exports`）、npm 專案初始化
- **需要的套件**：無（只使用 Node.js 內建功能）

## 具體任務

1. 在新資料夾中執行 `npm init -y`，建立 `package.json`
2. 建立 `mathUtils.js`，匯出以下四個函式：
   - `add(a, b)` — 加法
   - `subtract(a, b)` — 減法
   - `multiply(a, b)` — 乘法
   - `divide(a, b)` — 除法（若除數為 0，回傳字串 `"Error: 除數不能為零"`）
3. 建立 `index.js`，用 `require` 引入 `mathUtils.js`，並印出以下結果：
   - `10 + 3 = 13`
   - `10 - 3 = 7`
   - `10 * 3 = 30`
   - `10 / 3 = 3.3333333333333335`（保留原始浮點數即可）
   - `10 / 0 = Error: 除數不能為零`
4. 在 `package.json` 的 `scripts` 中加入 `"start": "node index.js"`，並用 `npm start` 執行

## 範例

```js
// mathUtils.js
function add(a, b) {
  return a + b;
}
function divide(a, b) {
  if (b === 0) return "Error: 除數不能為零";
  return a / b;
}
module.exports = { add, divide };

// index.js
const { add, subtract, multiply, divide } = require('./mathUtils');

console.log(`10 + 3 = ${add(10, 3)}`);
// 輸出：10 + 3 = 13

console.log(`10 / 0 = ${divide(10, 0)}`);
// 輸出：10 / 0 = Error: 除數不能為零
```

## 提示

- `module.exports` 可以匯出一個物件，同時包含多個函式
- 使用解構賦值 `const { add, subtract } = require('./mathUtils')` 可以只取出需要的函式
- `npm init -y` 的 `-y` 參數代表全部選項自動填入預設值，省去逐一確認的步驟
- 執行前先確認你的 Node.js 已正確安裝：`node -v`

## 學習重點

**跟後端 API 的關聯：**

真實的 Express API 專案幾乎每個檔案都在使用模組系統。例如：
- `routes/user.js` 匯出 router，在 `app.js` 中 `require` 引入
- `utils/db.js` 匯出資料庫連線，在各 controller 中共用
- `middleware/auth.js` 匯出驗證函式，掛載到需要保護的路由

今天練習的 `module.exports` + `require` 就是這整個架構的基礎骨架。

## 進階挑戰（選做）

1. **命名匯出 vs 預設匯出**：試著將 `mathUtils.js` 改成只用 `module.exports = function add(a, b) {...}` 匯出單一函式，觀察引入方式有何不同。
2. **建立第二個模組**：新增 `stringUtils.js`，包含 `capitalize(str)`（首字母大寫）和 `reverse(str)`（字串反轉），在 `index.js` 中一起使用。
3. **探索內建模組**：在 `index.js` 引入 Node.js 內建的 `os` 模組，印出目前作業系統名稱（`os.platform()`）和 CPU 核心數（`os.cpus().length`）。
