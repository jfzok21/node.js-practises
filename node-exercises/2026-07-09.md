# 第 1 週今日 Node.js 練習題：模組化計算機

## 題目說明

建立一個模組化的計算機程式，練習 Node.js 的模組系統（`require` / `module.exports`），並透過 `npm init` 初始化專案。你將把不同的數學運算拆分成獨立模組，再在主程式中組合使用。

## 技術要求
- 概念：`require`、`module.exports`、`npm init`、CommonJS 模組系統
- 需要的套件：無（純 Node.js 內建）

## 具體任務

1. 在專案目錄下執行 `npm init -y`，產生 `package.json`
2. 建立 `math.js`，用 `module.exports` 匯出以下四個函式：
   - `add(a, b)` — 加法
   - `subtract(a, b)` — 減法
   - `multiply(a, b)` — 乘法
   - `divide(a, b)` — 除法（若除數為 0，回傳字串 `"Error: 除數不能為 0"`）
3. 建立 `index.js`，用 `require` 引入 `math.js`，並對每個運算各呼叫一次，將結果印出到終端機
4. 用 `node index.js` 執行，確認輸出正確

## 範例

```js
// math.js
function add(a, b) { return a + b; }
function divide(a, b) {
  if (b === 0) return "Error: 除數不能為 0";
  return a / b;
}
module.exports = { add, divide /* ... */ };

// index.js
const math = require('./math');
console.log(math.add(10, 5));      // 15
console.log(math.subtract(10, 5)); // 5
console.log(math.multiply(10, 5)); // 50
console.log(math.divide(10, 5));   // 2
console.log(math.divide(10, 0));   // Error: 除數不能為 0
```

## 提示
- `module.exports` 可以匯出一個物件，把所有函式當成屬性放進去
- `require('./math')` 中的 `./` 代表「目前目錄」，不要省略
- 執行前確認你的終端機在正確的專案目錄下
- 可以用解構賦值讓引入更簡潔：`const { add, subtract } = require('./math')`

## 學習重點（跟後端 API 的關聯）

在真實的 Express / REST API 專案中，模組化是核心架構：
- **路由（routes）**、**控制器（controllers）**、**服務層（services）** 各自是獨立模組
- 每個模組只負責一件事（單一職責原則），讓程式碼易於測試與維護
- `require` / `module.exports` 是 Node.js 中串接這些模組的基礎語法
